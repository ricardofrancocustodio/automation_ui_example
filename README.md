# Test Automation solution using POM (Page Object Model) Design Pattern

 - A simple test automation solution for UI:
   * Access URL
   * Find Element 
   * Find and List Elements
   * Implicit wait
   * Explicit wait
  
  
  
  
 - Dependencies
   * Java 8
   * Maven
   * Junit
   * Selenium 
   * Gerkhin
   * Cucumber
   * Chromedriver
